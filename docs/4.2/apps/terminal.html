Terminal

simple terminal

Presentation​

A simple terminal emulator with a complete on-screen keyboard for Onion!

Usage​

Controls​

ButtonFunction
D-padNavigate left/right/up/down arrows
APress key
BToggle key
XShow / hide keyboard
YChange keyboard location (top/bottom)
L1Shift
R1Backspace
L2Left
R2Right
SelectTab
StartEnter
Menuquit (keyboard must be displayed)

Advanced​

Terminal is used to display many scripts in Onion like Scraper, OTA updates, PDF reader and video player launchers, ...
