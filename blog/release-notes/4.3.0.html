Official release notes for Onion V4.3.0

This release includes new systems, such as Nintendo DS and PICO-8, a lot of new and updated apps such as the Battery Monitor and the Blue light filter, support for the v4 Miyoo Mini's 560p screen as well as lots of other new features and fixes.

Features​

Core Onion features & apps​

Miyoo Mini v4 560p support​

by @tGecko

Support for the Miyoo Mini v4 with its 560p screen has been added.
The resolution is now automatically and 560p is the default resolution for RetroArch and NDS/Drastic on the Miyoo Mini v4. A system to allow ports to use the new resolution is also in place, but has to be added to the ports manually.

Screen recorder​

by @XK9274

A tool to record the screen, this tool will not offer perfect performance in games, but can be used to record/demonstrate apps and features.

Read More

VNC server​

by @XK9274

Remote display and remote control of your Miyoo Mini Plus from your computer thanks to VNC.

Read More

Blue light filter​

by @XK9274

Control the blue light filter schedule, intensity, or state.

Read More

PWM frequency adjustment​

by @tGecko

Added an option in Tweaks app to control the backlights PWM frequency. This can help reduce the buzzing sound coming from the speaker.

Read More

Battery Monitor​

by @Sichroteph

Onion Battery Tool enables to track and monitor historical battery charge while also providing an estimate of the remaining gaming time.

Read More

M3U playlist generator​

by @schmurtzm

This Tweaks app tool will generate automatically the m3u playlist files for multi-disc games.

Read More

App list sorting tool​

by @XK9274

This new tool included in Tweaks app will sort your Onion app list A-Z or Z-A.

Read More

Automatic selection of emulators in Package Manager​

by @aemiii91

Package Manager allows to auto select emulators that has roms present.

Read More

Mute background music option​

by @theUnBurn

Mute background music for themes.

Read More

Disable automatic time restore​

by @tGecko

Normally, Onion saves the system time at shutdown and restores it on bootup. This new option disables the restore part. To use it, create the file SD/.tmp_update/config/.noTimeRestore

Updated Filebrowser to 2.26​

by @edmunek and @XK9274

Update Filebrowser to 2.26 to fix CSS issue.

Read More

Profile-specific Themes, Favorites and Recent lists​

by @Sichroteph

Themes favorites and rencent lists are now associated to the current profile only (This was already the case for saves only). It allows to have your own configuration between your main profile and the guest profile.

Startup and checkoff scripts​

by @dragoonDorise

An easy way to run scripts at Onion's startup and just before powering off.

Pico-8 Wrapper (new standalone emulator)​

by @XK9274

An awesome port of the official Pico-8 engine. It offers an optimal compatibility and the native Pico-8 online store called "Splore". (In order to use the wrapper, you need to have previously purchased PICO-8)
